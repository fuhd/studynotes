数字
================================================================================
关于数字，Python的核心对象集合包括常规的类型：**整数**（没有小数部分的数字）、**浮点数**（概括
地讲，就是后边有小数部分的数字）以及 **更为少见的类型**（有虚部的复数、固定精度的十进制数、带分子
和分母的有理分数以及集合等）。

Python的基本数字类型还是相当基本的。Python中的数字支持一般的数学运行。例如，加号（`+`代表加法，
星号（`*`）表示乘法，**双星号（`＊＊`）表示乘方**。
```python
>>> 123 + 222
345
>>> 1.5 * 4
6.0
>>> 2 ** 100
1267650600228229401496703205376
```
