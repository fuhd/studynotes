Pycharm中配置Anaconda
================================================================================
在工作环境中我们会集成开发环境去编码，这里推荐JB公司的pycharm，而pycharm也能很方便的和anaconda
的虚拟环境结合。在 **File => Setting => Project:xxxx => Project Interpreter** 里面修改
`Project Interpreter`，点击齿轮标志，再点击`Add Local`，再选择某个环境的python.exe解释器即可。
