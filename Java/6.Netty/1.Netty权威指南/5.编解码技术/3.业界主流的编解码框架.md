业界主流的编解码框架
===========================================================
### Google的Protobuf
Protobuf全称Google Protocol Buffers，它由谷歌开源而来，在谷歌内部久经考验。它 **将数据结构以`.proto`
文件进行描述，通过代码生成工具可以生成对应数据结构的POJO对象和Protobuf相关的方法和属性**。

它的特点如下：
+ 结构化数据存储格式（XML，JSON等）；
+ 高效的编解码性能；
+ 语言无关，平台无关，扩展性好；
+ 官方支持Java，C++和Python三种语言；
