jinfo：Java配置信息工具
==========================================================
**`jinfo`的作用是实时地查看和调整虚拟机的各项参数**。使用`jps`命令的`-v`参数可以查看虚拟机启动时显式指定
的参数列表，但如果 **想知道未被显式指定的参数的系统默认值**，除了找资料外，就只能使用 **`jinfo`的`-flag`选项**
进行查询了。

### `jinfo`命令格式
```
jinfo  [option] <name>  pid
```
示例：
```powershell
jinfo -flag SurvivorRatio 17515
```
显示：
```
-XX:SurvivorRatio=8
```
