Oozie快速入门
================================================================================
## 1.构建Oozie

## 1.1.系统要求
+ Unix机箱（在Mac OS X和Linux上测试）
+ Java JDK 1.8+
+ Maven 3.0.1+
+ Hadoop 2.6.0+
+ Pig 0.10.1+

JDK命令（java，javac）必须位于命令路径中；Maven命令（mvn）必须位于命令路径中。

## 1.2.构建Oozie
从[Oozie站点](oozie.apache.org/)的“Releases”下拉菜单下载Oozie的源代码。
