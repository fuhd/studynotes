安装Hadoop
=================================================================================
### 下载并解压
从Hadoop官网下载hadoop-2.9.0.tar.gz，然后解压到/opt目录下：
```shell
$ sudo tar xzvf hadoop-2.9.0.tar.gz
$ sudo mv hadoop-2.9.0/ /opt/
```

### 改变hadoop文件的用户与组
将Hadoop文件的拥有者改为hadoop用户和组：
```shell
$ sudo chown -R hadoop:hadoop hadoop-2.9.0/
```
