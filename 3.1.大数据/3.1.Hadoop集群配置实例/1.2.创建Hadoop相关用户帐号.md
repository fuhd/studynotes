创建Hadoop相关用户帐号
=================================================================================
### 创建hadoop用户组
```shell
$ sudo groupadd -f hadoop
```

### 创建hadoop用户
```shell
$ sudo useradd -s /sbin/nologin -g hadoop hadoop
```

### 创建hdfs用户
```shell
$ sudo useradd -s /sbin/nologin -g hadoop hdfs
```

### 创建mapred用户
```shell
$ sudo useradd -s /sbin/nologin -g hadoop mapred
```

### 创建yarn用户
```shell
$ sudo useradd -s /sbin/nologin -g hadoop yarn
```
