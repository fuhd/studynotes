创建Hadoop相关用户帐号
=================================================================================
### 创建hadoop用户组
```shell
$ sudo groupadd -f hadoop
```

### 创建hadoop用户
注意，**不能使用 -s /sbin/nologin** ，否则不能从root切换到此帐户上。
```shell
$ sudo useradd -g hadoop hadoop
```
