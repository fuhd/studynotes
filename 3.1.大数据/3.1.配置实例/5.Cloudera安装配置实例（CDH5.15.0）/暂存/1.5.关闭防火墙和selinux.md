关闭防火墙和selinux
================================================================================
## 1.关闭防火墙
```shell
#停止firewall
systemctl stop firewalld.service
```

## 2.禁止firewall开机启动
```shell
systemctl disable firewalld.service
```

## 3.查看默认防火墙状态
```shell
# 关闭后显示notrunning，开启后显示running
firewall-cmd --state
```

## 4.关闭selinux
```shell
#编辑该文本
vim /etc/selinux/config
```
找到SELINUX改为：**SELINUX=disabled**
```
SELINUX=disabled
```
重启生效！！！
