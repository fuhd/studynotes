安装和配置数据库
================================================================================
## 1.下载MySQL5.7
```shell
$ curl -O https://cdn.mysql.com//Downloads/MySQL-5.7/mysql-community-server-5.7.22-1.el7.x86_64.rpm
```

## 2.安装MySQL5.7
```shell
$ sudo rpm -ivh mysql-community-release-el7-5.noarch.rpm
$ sudo yum update
$ sudo yum install mysql-server
```
