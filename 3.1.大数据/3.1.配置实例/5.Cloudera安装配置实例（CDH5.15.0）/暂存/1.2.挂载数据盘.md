挂载数据盘（可选）
================================================================================
## 查看是否有数据盘
```shell
$ fdisk -l
```

## 数据盘分区
如果是新盘（或已格式化的磁盘），需要对数据盘进行分区，例如：
```shell
$ fdisk /dev/vdb
```

## 新分区上创建一个文件系统
如果是新盘（或已格式化的磁盘），分区后还要为磁盘创建一个文件系统，例如：
```shell
#因为我是centos7，可以用xfs文件系统
$ mkfs.xfs /dev/vdb1
```

## 备份etc/fstab
```shell
$ cp /etc/fstab /etc/fstab.bak
```

## 向/etc/fstab写入新分区信息
```shell
#首先创建目录，例如：/data1
mkdir /data1
echo /dev/vdb1 /data1 xfs defaults 0 0 >> /etc/fstab
```

## 挂载文件系统
```shell
$ mount /dev/vdb1 /data1
```

## 查看目前磁盘空间和使用情况
```shell
$ df -h
```
