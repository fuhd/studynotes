配置Hive
=================================================================================
### 新建一个空的hive-site.xml文件
```shell
su - hadoop
$ cd /opt/apache-hive-1.2.2/conf/
$ touch hive-site.xml
```
**编辑hive-site.xml**：
```xml
<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<?xml-stylesheet type="text/xsl" href="configuration.xsl"?>
<configuration>

</configuration>
```
