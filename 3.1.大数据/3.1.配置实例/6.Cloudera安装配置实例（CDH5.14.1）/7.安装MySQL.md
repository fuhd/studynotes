安装MySQL
================================================================================
**CDH5.14.1** 支持MySQL5.7，那这里我们就选择 **MySQL5.7**。我们选择在 **server02** 上安装MySQL。

### 1.安装MySQL
**参见**：MySQL/来自互联网资料/在CentOS7中安装配置MySQL5.7.md

### 2.设置开机启动
```shell
chkconfig mysqld on
```

### 3.启动MySQL
```shell
systemctl start mysqld.serivce
```
查看状态：
```shell
systemctl status mysqld.serivce
```
