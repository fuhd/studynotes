安装MySQL
================================================================================
**CDH5.14.1** 支持MySQL5.7，那这里我们就选择 **MySQL5.7**。我们选择在 **server02** 上安装MySQL。

### 1.安装MySQL
**参见**：MySQL/来自互联网资料/在CentOS7中安装配置MySQL5.7.md

### 2.修改MySQL默认目录

#### 2.1.停止MySQL服务
```shell
systemctl stop mysqld
```

#### 2.1.建立目标存放目录
```shell
mkdir /data2
```

#### 2.2.拷贝文件
```shell
cp -R /var/lib/mysql /data2/
```

### 3.修改MySQL配置文件
编辑 **/etc/my.cnf**：
```shell
# 以前的配置用#进行注释了
datadir=/data2/mysql
socket=/data2/mysql/mysql.sock
```
