设置本地存储库
================================================================================
我们把存储库放到跟Ambari Server同一台主机上（`10.10.26.31` - server01）。

## 第一步：安装可选软件
```shell
yum install yum-utils createrepo
```

## 第二步：安装Web服务器

### 1.装Apache
```shell
yum install httpd
```

### 2.启动Apache
```shell
systemctl start httpd.service
```
