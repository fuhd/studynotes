卸载并清除Ambari和HDP
================================================================================
## 1.停止在Ambari中的所有服务

![停止在Ambari中的所有服务](img/31.png)

## 2.停止Ambari Server与Ambari Agent
在`server01.tythin.com`上执行：
```shell
ambari-server stop
```
在 **所有节点上** 执行：
```shell
ambari-agent stop
```

## 3.删除Ambari Server与Ambari Agent
在`server01.tythin.com`上执行：
```shell
yum remove ambari-server
```
在 **所有节点上** 执行：
```shell
yum remove ambari-agent
```

## 4.1.检查已安装服务

```shell
yum list installed | grep ambari
```
