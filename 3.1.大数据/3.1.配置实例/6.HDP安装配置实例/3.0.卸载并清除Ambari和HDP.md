卸载并清除Ambari和HDP
================================================================================
## 1.停止在Ambari中的所有服务

![停止在Ambari中的所有服务](img/31.png)

## 2.停止Ambari Server与Ambari Agent
在`server01.tythin.com`上执行：
```shell
ambari-server stop
```
在 **所有节点上** 执行：
```shell
ambari-agent stop
```

## 3.卸载Ambari Server与Ambari Agent
在`server01.tythin.com`上执行：
```shell
yum remove ambari-server -y
```
在 **所有节点上** 执行：
```shell
yum remove ambari-agent -y
```

## 4.删除已安装的服务组件
