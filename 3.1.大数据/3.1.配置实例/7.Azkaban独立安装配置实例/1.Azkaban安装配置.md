Azkaban安装配置
================================================================================
## 1.下载Azkaban源代码
从 [azkaban的github](https://github.com/azkaban/azkaban) 下载源代码。这里最新稳定版本
为：**azkaban-3.57.0.tar.gz**

## 2.解压缩Azkaban源代码包
```shell
tar -xzvf azkaban-3.57.0.tar.gz
```

## 3.从源构建
```shell
# 构建和安装发行版
./gradlew installDist
```

## 4.解压缩构建文件
```shell
tar -xzvf azkaban-3.57.0.tar.gz
```
