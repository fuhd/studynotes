配置主机名与网络名称
================================================================================
**这里以`bigdata005`的节点为例，其余的2个节点按相同的方式操作**。

## 1.设置主机名
```shell
$ sudo hostnamectl set-hostname bigdata005
```

## 2.编辑/etc/hosts
添加：
```
172.16.29.4 bigdata005
172.16.29.5 bigdata006
172.16.29.6 bigdata008
```

## 3.编辑/etc/sysconfig/network
添加：
```ini
NETWORKING=yes
HOSTNAME=bigdata005
```

## 4.重启所有节点 
```shell
reboot
```
