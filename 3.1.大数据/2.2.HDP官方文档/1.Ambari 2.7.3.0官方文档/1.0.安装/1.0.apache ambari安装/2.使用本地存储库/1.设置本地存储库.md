设置本地存储库
================================================================================
根据您的Internet访问权限，选择以下选项之一：
+ **不能访问网络**：此选项涉及下载存储库tarball，将tarball移动到集群中的选定镜像服务器，以及
解压缩tarball以创建
存储库。
+ 临时上网：此选项涉及使用临时Internet访问将软件包同步（使用重新同步）到所选镜像服务器以创建存
储库。
