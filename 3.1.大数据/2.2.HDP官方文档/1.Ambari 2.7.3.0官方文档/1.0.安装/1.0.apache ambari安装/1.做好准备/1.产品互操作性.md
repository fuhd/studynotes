本节介绍使用Ambari安装集群时应准备好的信息和材料。Ambari为您的集群提供端到端管理和监控解决方案。
使用`Ambari Web UI`和`REST API`，您可以从中心点部署、操作、管理配置更改，并监控群集中所有节
点的服务。
