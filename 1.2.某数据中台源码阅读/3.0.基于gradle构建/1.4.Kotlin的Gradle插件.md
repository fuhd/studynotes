Kotlin的Gradle插件
================================================================================
现在的最新版本是：**1.4.0（2020/09/01）**。

## 使用插件DSL
```groovy
plugins {
  id "org.jetbrains.kotlin.jvm" version "1.4.0"
}
```

## 旧版使用方法
```groovy
buildscript {
  repositories {
    maven {
      url "https://plugins.gradle.org/m2/"
    }
  }
  dependencies {
    classpath "org.jetbrains.kotlin:kotlin-gradle-plugin:1.4.0"
  }
}

apply plugin: "org.jetbrains.kotlin.jvm"
```
