单链表的整表删除
============================================================
当我们不打算使用这个单链表时，我们需要把它销毁，其实也就是在内存中将它释放掉，以便于留出空间给其他程序或软件使用。
单链表整表删除的算法思路如下:
1. 声明一结点`p`和`q`；
2. 将第一个结点赋值给`p`；
3. 循环：
  + 将下一结点赋值给`q`；
  + 释放`p`；
  + `q` 赋值给`p`；

实现代码算法如下：
```c
//初始条件：单链线性表L已存在，操作结果：将L重置为空表
Status ClearList(LinkList ＊L)
{
    LinkList p,q;
    p = (＊L) -> next;                     //＊p指向第一个结点
    while(p)                               //没到表尾
    {
        q = p -> next;
        free(p);
        p = q;
    }
    (＊L) -> next = NULL;                  //头结点指针域为空
    return OK;
}
```
