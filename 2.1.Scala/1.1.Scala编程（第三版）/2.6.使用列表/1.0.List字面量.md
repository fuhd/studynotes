List字面量
================================================================================
我们在前面的章节已经介绍过列表，一个包含原包'a'，'b'和'c'的列表写作`List('a', 'b','c')`。以
下是另外一些例子：
```scala
val fruit = List("apples", "oranges", "pears")
val nums = List(1, 2, 3, 4)
val diag3 =
  List(
    List(1, 0, 0),
    List(0, 1, 0),
    List(0, 0, 1)
  )
val empty = List()
```
**列表跟数组非常像，不过有两个重要的区别**。首先，**列表是不可变的**。也就是说，列表的元素不能
通过赋值改变。其次，**列表的结构是递归的（即链表）**，而数组是平的。
