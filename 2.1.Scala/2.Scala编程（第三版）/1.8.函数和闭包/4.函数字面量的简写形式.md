函数字面量的简写形式
================================================================================
Scala提供了多个省去冗余信息，更简要地编写函数的方式。

**一种让代码变得更简要的方式是略去参数类型声明**。示例：
```scala
val somNumbers = List(-11, -10, -5, 0, 5, 10)
someNumbers.filter((x: Int) => x > 0)
```
简写为：
```scala
someNumbers.filter((x) => x > 0)
```
