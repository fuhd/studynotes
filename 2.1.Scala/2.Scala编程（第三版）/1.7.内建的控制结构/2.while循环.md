while循环
================================================================================
Scala的while循环跟其他语言用起来没多大不同。它包含一个条件检查和一个循环体，只要条件检查为真，
循环体就会一遍接着一遍地执行。来看示例：
```scala
//用while循环计算最大公约数
def gcdLoop(x: Long, y: Long): Long = {
	var a = x
	var b = y
	while (a != 0) {
		val temp = a
		a = b % a
		b = temp
	}
	b
}
```
