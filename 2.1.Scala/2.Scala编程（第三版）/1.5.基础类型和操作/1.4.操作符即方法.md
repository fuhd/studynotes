操作符即方法
================================================================================
**操作符实际上只是普通方法调用的漂亮语法**。例如，`1 + 2`实际上跟`1.+(2)`是一回事。换句话说，
Int类包含了一个名为`+`的方法，接收一个Int参数，返回Int的结果。这个`+`方法是在你对两个Int做加
法时执行的：
```scala 
scala> val sum = 1 + 2
sum: Int = 3

scala> val sumMore = 1.+(2)
sumMore: Int = 3
```
