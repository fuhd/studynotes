算术操作
================================================================================
可以通过加法（`+`）、减法（`-`）、乘法（`*`）、除法（`/`）和取余数（`%`）的中缀操作符表示法对
任何数值类型调用算术方法。以下是一些示例：
```scala
scala> 1.2 + 2.3
res0: Double = 3.5

scala> 3 - 1
res1: Int = 2

scala> 'b' - 'a'
res2: Int = 1

scala> 2l * 3l
res3: Long = 6

scala> 11 / 4
res4: Int = 2

scala> 11 % 4
res5: Int = 3

scala> 11.0F / 4.0F
res6: Float = 2.75

scala> 11.0 % 4.0
res7: Double = 3.0
```
