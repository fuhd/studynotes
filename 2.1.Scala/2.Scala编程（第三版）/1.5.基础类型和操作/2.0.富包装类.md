富包装类
================================================================================
一些富操作：

| 代码 | 结果 |
| :------------- | :------------- |
| 0 max 5 | 5 |
| 0 min 5 | 0 |
| -2.7 abs | 2.7 |
| -2.7 round | -3L |
| 1.5 isInfinity | false |
| (1.0 /0) isInfinity | true |
| 4 to 6 | Range(4,5,6) |
| "bob" capitalize | "Bob" |
| "robert" drop 2 | "bert" |

上表中给出了一些例子。**这些方法可以通过隐式转换（`implicit conversion`）得到，关于隐式转换
的技巧，会在第21章详细介绍**。你当下需要知道的是，**本章提到的每个基础类型，都有一个对应的“富包
装类”，提供了额外的方法**。要了解基础类型的所有方法，你应该去看一下每个基础类型的富包装类的API
文档。
