隐式引入
===================================================================================
**Scala对每个程序都隐式地添加了一些引入**。本质上，这就好比 **每个扩展名为“.scala”的源码文件的顶部
都添加了如下三行引入子句**：
```scala
import java.lang._          //java.lang包的全部内容
import scala._                  //scala包的全部内容
import Predef._               //Predef对象的全部内容
```
