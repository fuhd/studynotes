表的删除和更新
==================================
### 表的删除
删除表的语法：
```sql
DROP TABLE <表名>;
```
示例：
```sql
DROP TABLE Shohin;
```
### 表定义的更新
语法：
```sql
ALTER TABLE <表名> ADD COLUMN <列的定义>;
```
**特定SQL**
```sql
--Oracle,SQL Server中不用写COLUMN
ALTER TABLE <表名> ADD <列的定义>;
--Oracle中同时添加多列
ALTER TABLE <表名> ADD (<列的定义>,<列的定义>,...);
```
