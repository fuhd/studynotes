MariaDB的默认数据库
============================================
运行mysql客户端程序，执行：
```sql
show databases;
```
显示：
```
MariaDB [mysql]> show databases;
+--------------------+
| Database           |
+--------------------+
| information_schema |
| mysql              |
| performance_schema |
| test               |
+--------------------+
4 rows in set (0.01 sec)
```
