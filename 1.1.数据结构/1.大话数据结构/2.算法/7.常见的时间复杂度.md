常见的时间复杂度
===========================================================
常见的时间复杂度，如下表：

| 执行次数函数 | 阶 | 非正式术语 |
| :--------- | :-- | :-------|
| 12 | **O(1)** | 常数阶 |
| 2n+3 | **O(n)** | 线性阶 |
| 3n2 + 2n + 1 | **O(n2)** | 平方阶 |
| 5log2n + 20 | **O(logn)** | 对数阶 |
| 2n + 3nlog2n + 19 | **O(nlogn)** | nlogn阶 |
| 6n3 + 2n2 + 3n + 4 | **O(n3)** | 立方阶 |
| 2n | **O(2n)** | 指数阶 |

常用的时间复杂度所耗费的时间从小到大依次是：
```
O(1) < O(logn) < O(n) < O(nlogn) < O(n2) < O(n3) < O(2n) 
也就是：
常数阶 < 对数阶 < 线性阶 < nlogn阶 < 平方阶 < 立方阶 < 指数阶
```
