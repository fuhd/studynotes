### 如何阅读本书
在这本书中，我们将描述许多`Linux`命令。每个命令的描述都有一个标准格式的标题。示例：

| 命令 | 选项 | 参数 |   |   |   |   |   |   |
|:----|:-----|:----|:--|:--|:--|:--|:--|:--|
|   |  |  | stdin | **stdout** | -file | **--opt** | **--help** | **--version** |
| ls | [options] | [files] |  |  |  |  |  |  |

上表显示了`ls`（列出文件）命令的标题，它用一种简单的格式描述了`ls`命令的一般用法：
```shell
ls [options] [files]
```
这意味着你 **必须首先输入"`ls`"，是否提供选项及文件名可以视情况而定**。方括号 **“[”和“]”** 并不需要实际
进行输入，**它们只是用来强调其所包含的内容是可选的**。
