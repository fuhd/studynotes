### 如何阅读本书
在这本书中，我们将描述许多`Linux`命令。每个命令的描述都有一个标准格式的标题。示例：

| 命令 | 选项 | 参数 |   |   |   |   |   |   |
|:----|:-----|:----|:--|:--|:--|:--|:--|:--|
|   |  |  | stdin | **stdout** | -file | **--opt** | **--help** | **--version** |
| ls | [options] | [files] |  |  |  |  |  |  |
