组管理
=================================================================================
| 命令 | 说明 |
| :------------- | :------------- |
| **groups** | 打印用户所属组 |
| **groupadd** | 创建组 |
| **groupdel** | 删除组 |
| **groupmod** | 修改组 |

**组（group）代表一群用户的共同身份。如果你将执行某项动作（如修改文件）的权限赋予某个组，那么该组的所
有用户将拥有此权限**。

**只有root可以通过编辑/etc/group文件来改变组成员**。若要更改某文件的组拥有权，请参见“文件属性”中的
**chgrp命令**。
