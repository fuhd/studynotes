Apache JMeter概述
==============================================================
Apache JMeter是一个100%的纯Java桌面应用，用于 **压力测试** 和 **性能测量**。
它最初被设计用于web应用测试，但后来扩展到其他测试领域。Apache JMeter可以用于对 **静态** 的和 **动态** 的资源
（文件、Servlet、Web动态语言、Java对象、数据库和查询、FTP服务器等）的性能进行测试。

### Apache JMeter所支持的协议
+ Web: HTTP、HTTPS
+ SOAP
+ FTP
+ JDBC
+ LDAP
+ 面向消息的中间件（通过JMS）
+ 邮件：SMTP(S)、POP3(S)、IMAP(S)
+ MongoDB(NoSQL)
+ TCP
+ 原生命令或者shell脚本
