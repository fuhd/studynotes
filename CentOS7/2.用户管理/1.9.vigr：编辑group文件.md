vigr：编辑group文件
==========================================================
`vigr`命令可以直接编辑`/etc/group`文件。

常用选项有：

| 选项 |  |
| :------------- | :------------- |
| -g,--group | 编辑group数据库 |
| -p,--passwd | 编辑passwd数据库 |
| -s,--shadow | 编辑shadow或gshadow数据库 |

### 示例：修改group文件
```powershell
$ sudo vigr 
```
或者：
```powershell
$ sudo vigr -g
```

### 示例：修改passwd文件
```powershell
$ sudo vigr -p
```
