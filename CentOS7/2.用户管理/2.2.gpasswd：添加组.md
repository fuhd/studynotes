gpasswd：添加组
======================================================
`gpasswd`命令用来将用户添加到组。

常用选项：

| 选项 |  |
| :------------- | :------------- |
| -a,--add USER | 向组GROUP中添加用户USER |
| -d,--delete USER | 从组GROUP中删除用户 |
| -M,--members USER,...|设置组Group的成员列表|
| -A,--administrators ADMIN,...|设置组的管理员列表|

### 示例：向组groupA中添加用户user1
```powershell
$ sudo gpasswd -a user1 groupA
```
### 示例：从组groupA中删除user1
```powershell
$ sudo gpasswd -d user1 groupA
```
