userdel：删除用户
========================================================
`userdel`命令用来删除一个用户。

下面是最常用的选项：

| 选项 |   |
| :------------- | :------------- |
| -r,--remove  | 删除主目录和邮件池  |
| -Z,--selinux-user | 为用户删除所有的SELinux用户映射 |

### 示例：删除一个用户
```powershell
$ userdel -r fhdA
```
此命令删除用户fhdA在系统文件（`/etc/passwd`，`/etc/shadow`，`/etc/group`）中的记录，
同时删除用户的主目录。
