su：切换身份
==========================================================
`su`命令用来让用户暂时变更登入身份。用法：`su [选项] [-] [USER[参数]...]`。

常用选项：

| 选项 |  |
| :------------- | :------------- |
| -,-l,--login | 使shell成为登录shell，也就是说，完全换切到用户环境中 |
| -c,--command <命令> | 使用 -c 向shell传递一条命令 |

### 示例：变更帐号为`root`并在执行`ls`指令后退出变回原使用者
```powershell
$ su -c 'ls /etc' root
```

### 示例：完全切换到`root`帐户下
```powershell
$ su - root
```
