groupadd：添加组
========================================================
groupadd命令用于将新组添加到系统中，使用的是超级用户权限。

常用选项是：

| 选项 |  |
| :------------- | :------------- |
| -g,--gid GID | 为新组使用GID |
| -f,--force | 如果组已经存在，则成功退出。并且如果GID已经存在，则取消-g |

### 示例：创建一个新组，设置组ID
```powershell
$ sudo groupadd-g 744 fhdB
```
