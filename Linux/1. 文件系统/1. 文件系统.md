主目录
=======================================================
Linux里每个用户都有一个存入个人文件的目录，称为 **主目录**。普通用户的个人目录通常位于 **/home** 目录下，
典型的目录名称是 **/home/your-username**等等，对于超级用户来说，主目录是 **/root**。

有几种方法可以找到或者引用你的主目录。

### cd
如果不带参数，cd命令会带你移动到你的主目录。

### HOME环境变量
环境变量 **HOME** 包含你的主目录名称。示例： 
```shell
$ echo $HOME 
```
显示：
```
/home/fuhd
```
