### 第八题
表达式`1 == 1 >>> 32`的值是：（**true**）
```
提示：

>>>运算符的优先级高于==
```
```
提示：

1 >>> 32 == 1 与 1L >>> 64 == 1，相当于没有位移。

对于int类型,只有32位,右移32位等于右移0位
对于long类型,有64位,右移64位等于右移0位

原因：
根据java规范中描述 int a<<b, b这个操作数只能取二进制数的低五位（就是最后5位）；b操作数取后五位，也可以理解成
b/32取余数，所以 int a << b 可以理解成 int a << (b%32)
```
