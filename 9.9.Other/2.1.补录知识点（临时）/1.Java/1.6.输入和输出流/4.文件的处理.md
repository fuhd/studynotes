文件的处理
================================================================================
File对象可以用来 **生成与文件（及其所在的路径）或目录结构相关的对象**。不同的系统可能会有不同的
目标结构表示法，**但使用File类可以达到与系统无关的目的**。

## File类
要创建一个新的File对象可以使用以下3种构造方法：

### 第1种
```java
File myFile;
myFile = new File("mymotd");
```

### 第2种
```java
myFile = new File("/", "mymotd");
```

### 第3种
```java
File myDir = new File("/");
myFile = new File(myDir, "mymotd");
```
