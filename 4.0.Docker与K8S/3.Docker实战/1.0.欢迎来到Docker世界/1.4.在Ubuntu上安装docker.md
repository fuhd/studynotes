在Ubuntu上安装docker
===================================================================================
## 1. 先决条件

### 1.1. 操作系统要求
要安装 **Docker Engine-Community**，您需要以下Ubuntu版本之一的 **64位版本**：
+ Disco 19.04
+ Cosmic 18.10
+ Bionic 18.04 (LTS)
+ Xenial 16.04 (LTS)

Docker Engine - Community在 `x86_64`（或`amd64`）、`armhf`、`arm64`、`s390x`（IBM Z）和
`ppc64le`（IBM Power）架构上得到支持。
