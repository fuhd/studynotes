证书使用
=====================================================================
**`java`提供了完善的数字证书管理实现**，我们几乎无需关注相关具体算法，**仅通过操作密钥库和数字证书就可完成
相应的加密/解密和签名/验证操作。密钥库管理私钥，数字证书管理公钥，私钥和公钥分属消息传递两方，进行加密消息传递**。

因此，**我们可以将密钥库看做私钥相关操作的入口，数字证书则是公钥相关操作的入口**。
