原子性、内存可见性和重排序——重新认识synchronized和volatile
===============================================================================
原子（`Atomic`）操作指相应的操作是单一不可分割的操作。例如，对`int`型变量`count`执行`count++`的操作
就不是原子操作。这是因为`count++`实际上可以分解为`3`个操作：
1. 读取变量`count`的当前值；
2. 拿`count`的当前值和`1`做加法运行；
3. 将`count`的当前值增加`1`后的值赋值给`count`变量；
