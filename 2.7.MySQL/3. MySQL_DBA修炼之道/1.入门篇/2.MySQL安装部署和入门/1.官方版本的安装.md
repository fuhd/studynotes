### 1.通过yum源安装MySQL（CentOS7）
见：MySQL/来自互联网资料/1.在CentOS7中安装配置MySQL5.7.md

| 目录 | 目录内容 |
| :----| :------|
| /usr/bin | 客户端程序和脚本 |
| /usr/sbin | mysqld服务器 |
| /var/lib/mysql | 日志文件，数据库文件 |
| /usr/share/mysql | 错误消息和字符集文件 |
| /etc/my.cnf | 配置文件 |
