存储引擎简介
================================================================================
运行如下命令可查看表的引擎：
```sql
use demodb;
show table status like 'user';
```
```
'user', 'InnoDB', '10', 'Dynamic', '0', '0', '16384', '0', '0', '0', NULL, '2018-01-24 11:33:23',NULL, NULL,
'utf8_general_ci', NULL, '', ''
```
