Atom中配置Rust开发环境
===============================

### 安装language-rust插件
安装Rust插件（`language-rust`），它提供 **代码高亮** ！步骤如下：
```
Edit -> Preferences -> install -> install packages -> 安装language-rust插件
```  

### 安装racer插件
安装Rust插件（`racer`），它提供 **代码自动补全** !

### 安装linter-rust插件
安装Rust插件（`linter-rust`），它提供 **rustc和cargo支持** !

### 安装atom-beautify插件
安装Rust插件（`atom-beautify`），它提供 **美化支持** ！
