逐步求解问题：局部定义
=======================================
**函数定义中的局部定义：这些局部定义可能是求解问题中有用的函数，或者有用的中间值。**

### 例子
在模块`PicturesSVG.hs`中定义一个有关图形的函数，假如函数的输入是马的图形，那么函数输出图示的图形
（翻转并反色）。

解决该问题的一种方法是：
```haskell
fourPics :: Picture -> Picture
fourPics pic =
  left `beside` right
    where
    left = ...
    right = ...
```
